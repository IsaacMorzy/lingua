# Issue Tracker

> Where work is tracked for the `lingua` repository.

## Tracker

GitHub Issues in `IsaacMorzy/lingua`.

## Tooling

Use the `gh` CLI for creating, updating, and linking issues.

## PRs as request surface

External PRs are treated as issues with attached code. The `triage` skill applies the same roles and states to PRs.

## Agent permissions

Agents may create issues, add them to the IJLAPS Website project board, and update labels. Agents should record significant tracker actions in `loop-run-log.md`.
